IMAGES REQUISES POUR L'APPLICATION
===================================

Crée les images suivantes et place-les dans ce dossier:

1. icon.png
   - Taille: 1024x1024 pixels
   - Format: PNG (avec transparence possible)
   - Icône principale de l'app

2. splash.png
   - Taille: 1284x2778 pixels (ou similaire)
   - Format: PNG
   - Écran de démarrage

3. adaptive-icon.png (Android uniquement)
   - Taille: 1024x1024 pixels
   - Format: PNG avec transparence
   - Icône adaptative Android

TU PEUX:
- Utiliser Canva (canva.com) pour créer ces images gratuitement
- Utiliser un générateur d'icônes comme appicon.co
- Demander à un designer

CONSEIL:
Pour le logo, tu peux utiliser un simple cercle avec les initiales "FT"
ou un symbole représentant le temps/les amis (horloge, personnes, etc.)
